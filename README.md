# Web3 Encrypted Chat DApp (Alpha)

An Ethereum-based chat application inspired by Telegram, 100% Web3. Users interact using wallet addresses onlyâ€”no usernames or centralized moderation. All messages are encrypted and stored on-chain.

## Features
- Private DM and group creation (on roadmap)
- Phantom and MetaMask login
- No usernames; only addresses
- Message deletion by sender
- 100% freedom: no moderation
- Blockchain-secured

## Setup
1. `npm install`
2. Deploy contract to Ethereum testnet
3. Run frontend: `npm run dev`
